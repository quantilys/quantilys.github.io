--- layout: page title: Home --- Jekyll Now illustrates the easiest possible workflow for setting up a Jekyll powered blog. My goal is for you to be able to use it to set up your blog as quickly as possible—giving you more time to spend on writing epic blog posts! ## Features ✓ _Command-line free fork-first workflow_, using GitHub.com to create, customize and post to your blog ✓ Fully responsive and mobile optimized base theme ([Theme Demo](http://jekyllnow.com)) ✓ Sass/Coffeescript support using Jekyll 2.0 ✓ Free hosting on your GitHub Pages user site ✓ Markdown blogging ✓ Syntax highlighting ✓ Disqus commenting ✓ Google Analytics integration ✓ SVG social icons for your footer ✓ 3 http requests, including your avatar ✘ No installing dependancies ✘ No need to set up local development ✘ No configuring plugins ✘ No need to spend time on theming ✘ More time to code other things ... wait ✓! ## That 2–3 minute set up Check out the [Quick Start in the Jekyll Now repository](http://github.com/barryclark/jekyll-now#quick-start).
